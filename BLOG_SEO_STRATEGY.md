## ✅ Quality Assurance Checklist

Vor Veröffentlichung jeden Artikels überprüfen:

- [ ] **Meta Tags**
  - Title: 50-60 Zeichen
  - Description: 150-160 Zeichen
  - H1 enthält Haupt-Keyword
  
- [ ] **Semantic HTML**
  - H1 → H2 → H3 Struktur logisch
  - `<article>` wraps Gesamter Artikel
  - `<section>` für thematische Blöcke
  
- [ ] **Schema Markup**
  - JSON-LD "Article" Schema
  - FAQ Schema für alle Q&A-Sections
  - BreadcrumbList für Navigation
  
- [ ] **Images**
  - Beschreibende Alt-Texte mit Keywords
  - Optimierte Größen (< 100KB)
  - Responsive Srcsets
  
- [ ] **Performance**
  - Core Web Vitals grün
  - Pagespeed > 85
  - Gzipped CSS/JS

## 🚀 Next.js Migration (Zukünftig)

Sobald auf Next.js migriert, nutzen Sie die **Metadata API**:

```typescript
// app/ratgeber/[slug]/page.tsx

import { Metadata } from 'next';

export async function generateMetadata({ params }): Promise<Metadata> {
  const slug = params.slug;
  const article = await fetchArticleBySlug(slug);
  
  return {
    title: article.title,
    description: article.description,
    openGraph: {
      title: article.ogTitle,
      description: article.ogDescription,
      type: 'article',
      url: `https://dppflash.de/blog/ratgeber/${slug}`,
      authors: ['Julian Gräfen']
    },
    twitter: {
      card: 'summary_large_image',
      title: article.twitterTitle,
      description: article.twitterDescription
    }
  };
}

export default function ArticlePage({ params }) {
  return <ArticleComponent slug={params.slug} />;
}
```
